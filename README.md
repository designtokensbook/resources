# Design token book

A website for the Design tokens book.

![open graph](public/og.png)

## Running Locally

1. Install dependencies using pnpm:

```sh
pnpm install
```

2. Start the development server:

```sh
pnpm dev
```

3. Build the production bundle:

```sh
pnpm build
```
